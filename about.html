ABOUT


The project is an authentication system that validates users for accessing the system only when they have input the correct password. The project involves three levels of user authentication. There are varieties of password systems available, many of which have failed due to bot attacks while few have sustained it but to a limit. In short, almost all the passwords available today can be broken into a limit.

Hence, we decided to design a project aimed to achieve the highest security in authenticating users. It contains three logins having three different kinds of password systems. The password difficulty increases with each level, and users have to input the correct password for a successful login. Users are given the privilege to set passwords according to their wishes.

The project comprises a text password (i.e., passphrase), a color-based pattern password, and a one-time password (OTP) sent to their registered email/mobile numbers for the three levels respectively. This way, there would be negligible chances of bots or anyone cracking the passwords. Even if they have cracked the first or second level, it would be impossible to crack the third one.

Many users find the most widespread text-based password systems unfriendly. So, in the case of the three-level password system, we tried to create a simple user interface and provide users with the best possible comfort in securing their passwords.
